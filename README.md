# AI-Coder

## Arena Duel — 3D FPS

A browser-based first-person arena shooter: you versus one AI bot.

### Controls

- **↑ / ↓** — Move forward / backward
- **← / →** — Turn left / right
- **Space** — Shoot

### Run

**Cách đơn giản:** double-click `index.html` (cần `three.min.js` cùng thư mục — đã có sẵn trong repo).

**Hoặc** dùng server local:

```bash
python3 -m http.server 3456
```

Mở http://localhost:3456 và bấm **Bắt đầu / Start**.
